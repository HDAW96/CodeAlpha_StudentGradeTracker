# 🎓 Student Grade Tracker

A simple Java console application developed as part of the CodeAlpha Java Programming Internship.

The program allows users to enter student names and grades, then generates a performance summary including average, highest, and lowest grades.

---

## 📌 Features

- ✅ Add multiple students and their grades
- ✅ Store data using ArrayLists
- ✅ Display all entered students and grades
- ✅ Calculate the average grade
- ✅ Find the highest grade
- ✅ Find the lowest grade
- ✅ Basic exception handling for invalid inputs
- ✅ Console-based user interface

---

## 🛠 Technologies Used

- Java
- Java Collections Framework (ArrayList)
- Scanner Class
- Exception Handling

---

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/HDAW96/CodeAlpha_StudentGradeTracker.git
```

### 2. Navigate to the project folder

```bash
cd CodeAlpha_StudentGradeTracker
```

### 3. Compile the program

```bash
javac Driver.java
```

### 4. Run the application

```bash
java Driver
```
